### Troubleshooting

**Issue:** Visual Engine not connecting
```bash

python visual_engine/backend/victor_visual_server.py

# Then run runtime in another terminal
python victor_interactive.py
```

**Issue:** Tensor Core unavailable
```bash

# Or continue without - quantum mesh still works with NumPy
```

**Issue:** Session file permissions
```bash
# Ensure logs directory is writable
chmod -R 755 logs/
```
